    Validator.Support Class

    Defined in: src/ember-validator.js:231
    Module: Ember.Validator

    validate

    () Ember.Validator.Results

    Defined in src/ember-validator.js:381

    Runs all validations defined in the validations object and stores results. The method returns a results object.

    Returns:

    Ember.Validator.Results:

    an instance of Validator.Results

    Related: Validator.Results, validations

    Example:

    Getting the validation

    var person = App.Person.create({ name: null });

person.validate().get('isValid') // false;;

    Getting the error message

    person.validate().getMsgFor('name'); // 'name is required'
